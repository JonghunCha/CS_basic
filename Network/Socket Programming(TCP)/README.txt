서버 코드와 클라이언트 코드 2개로 이루어져 있다.

리눅스 환경에서 구현하였으며 가상 머신을 이용해 서로 다른 ip주소를 가진 환경에서 테스트를 진행하였다.

별 다른 기능을 구현한 것은 없지만 서버-클라이언트간 통신을 직접 구현해 본 코드이다.

서버 코드는 컴파일 후 실행을 시키면 에러가 나거나 강제종료를 하기 이전에는 계속해서 클라이언트의 연결 요청을 기다린다.

클라이언트 코드는 서버측과 연결이 되면 "Hello Wolrd"라는 데이터를 버퍼에 보낸 후 연결을 종료하도록 구현하였다.

서버는 클라이언트로 받은 데이터를 printf로 터미널에 출력한 뒤 연결을 종료하도록 구현하였다.
